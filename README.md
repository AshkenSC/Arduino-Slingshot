# Arduino-Slingshot

里贾纳大学CS207课程项目。实现了一个基于Arduino的声控投石机。

通过操纵杆调整投石机方向；按下按钮进入射击模式，装弹上膛；声控发射。

小组成员：Horst Fuchs; Mohammad; Shicheng Ai
